# Alura-Store-Data-Challenge---Miguel-APK23
Repositorio con la solución al Challenge de análisis de datos de Alura Store, donde se procesan y visualizan datos de ventas de varias tiendas para generar insights sobre ingresos totales, ventas por categoría y comparativas entre tiendas. Incluye notebooks en Python con pandas y visualizaciones con Matplotlib y Seaborn.
